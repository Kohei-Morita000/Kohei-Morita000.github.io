# 長編化設計 01：最後のエレベーター

## 目標

- 本文量：日本語で約4,000〜5,000字（約1,100〜1,300語相当）
- 読了時間：約8分
- ジャンル：意味怖／オフィス怪談
- 中心テーマ：見て見ぬふりをした責任、会社による事故隠蔽、終わらない点検
- 読後感：一度は鎮まったように見せ、最後の一文で循環する恐怖へ反転させる

## 登場人物

### 佐伯直人（語り手・32歳）

17階に入るIT会社の社員。責任感が強い一方、追い詰められると確認作業を省略する癖がある。三年前の記憶に一部欠落があり、本人は過労で倒れたためだと思っている。

### 黒田（夜間警備員・58歳）

建物の事故履歴を知る警備員。口数は少ないが、監視映像や設備記録を細かく保存している。会社側が事故記録を削除したことに不信感を持っている。

### 村瀬（故人・設備員・当時46歳）

三年前、18階の機械室付近で転落死した設備員。公式には単独作業中の事故とされているが、実際には故障したエレベーターから直人を逃がすために残った。

## 物語の核

直人は三年前、エレベーターの安全装置に異常があるという村瀬の警告を、納期を優先して十分に確認せず「問題なし」と処理した。その夜、直人自身が故障したエレベーターに閉じ込められ、村瀬に救出された。会社は重大な管理責任を隠すため、直人の記憶の混乱を利用し、事故の詳細を伏せた。

事故から三年後の同じ夜、直人は再び17階で一人残業する。18階を示す空のエレベーターは、未完了の点検と隠された記憶を回収するために現れる。

## 起承転結

## 起：日常と最初の違和感（約250〜300語相当）

1. 直人は大規模なシステム移行のため、17階のオフィスに一人残る。
2. 日付が三年前の設備員死亡事故と同じ日であることを、社内カレンダーの小さな追悼表示で知る。
3. 18階は機械室だけで、通常のエレベーターボタンには存在しないと説明する。
4. 午前1時48分、無人のエレベーターが17階へ到着する。
5. 行先表示には「18」。箱内には機械油と濡れたコンクリートの匂いが残る。
6. 扉が閉まる直前、鏡に作業服の肩だけが映る。

### 起で置く伏線

- 直人の腕時計が、エレベーター到着後だけ8分遅れる。
- 三年前の事故当夜も、直人は会社にいた可能性を示す曖昧な同僚の発言。
- 直人の社員証の角に、本人の覚えがない深い傷がある。

## 承：異変の反復と過去の発掘（約400〜450語相当）

1. エレベーターは十数分おきに戻り、毎回18階を示す。
2. 社内チャットへ、削除済みのアカウント「村瀬設備」から「点検が終わっていません」と届く。
3. 直人が返信すると、「三年前、あなたは中にいた」と返る。
4. 過去メールを検索し、村瀬から安全装置異常の報告が届いていたことを発見する。
5. そのメールには直人名義で「問題なし。予定どおり運転してください」と返信されている。
6. 直人は送信した記憶がないが、メール時刻は事故直前だった。
7. 電話越しの同僚が、前夜に直人らしき人物を18階の監視画面で見たと言う。
8. オフィスの照明が非常灯だけになり、廊下の奥から工具箱を引きずる音が近づく。
9. エレベーターが再び開き、箱の奥から「まだ乗らないの？」と聞こえる。
10. 直人は逃げたいのに、右手だけが勝手に箱内へ向かって手招きを始める。

### 承で強める心理

- 直人は事故への関与を否定しながら、メールを消したい衝動に駆られる。
- 「覚えていない」のではなく「思い出さないようにしていた」可能性を徐々に示す。
- 会社の納期優先体質と、直人自身の小さな判断が事故につながったことを描く。

## 転：監視映像と記憶の反転（約300〜350語相当）

1. 直人は非常階段で地下警備室へ逃げる。
2. 黒田が17階の監視映像を確認する。
3. 映像では、直人が空のエレベーターへ何度も手招きしている。
4. 直人の背後には、作業服姿の男が密着するように立っている。
5. 黒田は男を村瀬だと認め、保存していた三年前の非公式映像を見せる。
6. 三年前、故障したエレベーター内には直人がいた。村瀬は非常停止装置を手動で押さえ、直人を17階へ押し出した後、昇降路へ落下した。
7. 直人は、自分が事故の直前に安全装置の警告を無視した記憶を取り戻す。
8. 会社は直人に「過労で倒れた」と説明し、事故映像とメールを削除した。
9. 現在の監視映像で村瀬が直人の耳元へ口を寄せる。同時に警備室のスピーカーから「確認を終わらせろ」と流れる。

## 結：未完了の点検と循環（約200〜250語相当）

1. 黒田は保守用キーで18階を開け、直人とともに機械室へ向かう。
2. 機械室には、三年前に紛失した直人の社員証と、村瀬の潰れたヘルメットが並べられている。
3. 直人は事故報告の隠蔽と自分の責任を認め、翌朝すべて公表すると誓う。
4. 異音が止まり、エレベーターの表示が18から17へ戻る。
5. 直人は地上へ戻れたと思い、会社の内部告発窓口へ資料を送信する。
6. 翌朝、黒田が監視記録を確認すると、警備室へ来た直人の姿は映っていない。
7. 17階の映像には、直人が一人でエレベーターへ乗り、存在しない18階へ上がる様子だけが残っている。
8. その夜、別の残業社員の前でエレベーターが開く。
9. 箱の奥には村瀬と直人が並び、直人が右手で手招きする。

### 最後の一文候補

> 「まだ乗らないの？」と尋ねた声は、今度は間違いなく私のものだった。

## 伏線と回収

| 伏線 | 回収 |
|---|---|
| 腕時計が8分遅れる | 三年前、直人が救出されてから村瀬が落下するまでの時間 |
| 社員証の深い傷 | 故障した扉へ挟まれた際についた傷 |
| 機械油と濡れたコンクリートの匂い | 村瀬が死亡した昇降路と機械室の環境 |
| 作業服の肩だけが鏡に映る | 村瀬が直人の背後に立っている |
| 直人の無意識の手招き | 終盤で直人自身が次の乗客を招く側になる |
| 「点検が終わっていません」 | 設備点検だけでなく、事故責任の確認と告白を意味する |

## 執筆上の制約

- 怪異の説明をすべて断定せず、監視映像と直人の記憶に若干の矛盾を残す。
- 会社批判だけに寄せず、直人個人の判断と罪悪感を物語の中心に置く。
- 中盤まで村瀬を単純な加害霊に見せ、転で救助者だったと反転させる。
- 残酷描写は抑え、音、匂い、監視映像、身体の不随意動作で恐怖を作る。
- 既存の最後の台詞を活かしながら、長編版では循環構造に発展させる。
