## 起承転結の確認

### 起

- 障害対応で最終便を逃した会社員・大庭裕也を語り手に設定
- 妻・沙織と幼い息子・蓮への帰宅連絡を置き、トンネル内の呼び声に応じやすい感情的な根拠を作った
- 時刻表にない古いバス、反応しないIC端末、運賃を取らない運転手から通常便ではない違和感を開始
- 制服姿の高校生、濡れた作業着の老人、赤い傘の女性を異なる年代の乗客として配置
- 運転席横の乗車表へ裕也の名前が現れ、乗車した時点で怪異の記録へ登録されたことを先行提示
- 車内時計が一分戻る描写により、通常の時間進行から外れた運行であることを示した

### 承

- 高校生・水野莉子が母親の声へ返事をすると、降車ボタンが自動点灯して扉が開く規則を提示
- 作業着の永井が自宅と思い込んだ場所で降車し、本人がボタンを押すことも返事と同じ行為だと示した
- 赤い傘の女性の「前にもここで降りました」から、降車した人物が別の周回で車内へ戻る場合があることを示唆
- 車内表示が「終点」になった後も山道を走り続け、短編版の状況へ自然に接続
- 同じ水守トンネル口を通過するたび白い列が増え、降りた莉子と永井が列へ加わることで降車の結果を可視化
- 空席の濡れ、無人で点灯する降車ボタン、乗客名を読む放送を用い、死者が車内へ近づく段階を作った
- 乗車表の日付から、最初に見た三人が同じ夜の乗客ではないことを明示
- 裕也の欄に家族構成まで記載し、後の呼び声が乗車表と記憶を利用している可能性を補強

### 転

- 水守トンネルへ入ると車内灯が消え、短編版の濡れた足音と運転手の警告を維持
- 母、妻、息子の声を、裕也が実際に返事をしやすい順序で配置
- 乗車直前の妻とのメッセージを根拠に、裕也が呼び声を記憶の模倣だと判断して耐える流れを作った
- 運転手・佐久間も娘の声に呼ばれ、生者だけでなく運転手も同じ規則へ拘束されていることを示した
- 自分自身の声による「席を代わって」が後方席ではなく運転席から聞こえることで、交代対象が運転席だと示した
- 乗車表の裕也の欄へ「交代待ち」が追加され、返事をしなくても次の運転手候補として登録されたことを明確化
- 事故当夜の衝撃音だけを再現し、具体的な遺体描写を使わず事故の残響を表現
- 裕也と佐久間の双方が返事を拒み、夜明けまで運行を継続することで脱出条件を成立させた

### 結

- 朝の営業所、空の運転席、廃止路線、十年前の事故という短編版の結末を維持
- 事故車両が営業所へ戻っていないことと、防犯カメラにバスが映らないことから、通常の車両ではないと確定
- 莉子と永井が異なる年の行方不明者と一致し、車内で年代が重なっていた設定を現実側の記録で補強
- 古い乗車券の裏へ「交代待ち」が転写され、怪異が営業所で終わっていないことを示した
- 一年間バスを避けた裕也の前へ同じ便が再来し、佐久間が運転席から乗客席へ移っている反転を配置
- 空の運転席と裕也自身の車内放送により、佐久間から裕也への交代が始まったことを示した
- 最終文「乗務員交代のため、発車します」で、短編版の「席を代わって」を運転席の交代要求として回収

## 伏線回収

| 伏線 | 回収状況 |
|---|---|
| 反応しないIC端末と不要な運賃 | 通常の営業便ではなく、乗車表だけで乗客を管理する事故後の便として回収 |
| 乗客の服装や所持品の年代差 | 三人が異なる年に同じ最終便へ迷い込んだ人物であると乗車表で回収 |
| 老人の濡れた作業着から水滴が落ちない | 生身の乗客ではなく、事故や過去の雨をまとった状態で運行へ残る人物だと示した |
| 赤い傘の女性の「今日も人数が違う」 | 女性が複数回の運行を経験し、一度降車した後に再乗車している人物として回収 |
| 名前を読む車内放送 | 名前への返事を降車意思として扱う怪異の規則として回収 |
| 白い列が通過ごとに増える | 呼び声に応じて降車した乗客が次の周回から列へ加わるためと判明 |
| 「次で降りると戻れません」 | 通常の下車ではなく、白い列側へ移されて次の便を待ち続けることを意味した |
| 乗車表の「佐久間修一　乗務継続」 | 佐久間が死後十年間、生者との交代を拒んで運転してきた記録として回収 |
| 自分自身の声の「席を代わって」 | 将来の裕也が運転席へ座り、佐久間と乗務交代することを求める声として回収 |
| 裏面に転写された「交代待ち」 | その夜に返事をしなくても、裕也が次回の運転手候補として残ったことを示した |
| 翌年の空の運転席 | 佐久間が乗客席へ移り、裕也の乗務開始が準備された結末として回収 |

## 文章監査

- 短編版の終点表示、反復する停留所、白い列、トンネル、濡れた足音、家族の声、空の運転席、廃止路線をすべて維持した
- 短編版の「席を代わって」を削除せず、長編版では乗客席から運転席へ意味を拡張した
- 運転手を単純な加害者にせず、生者を犠牲にする交代を十年間拒んだ人物として描いた
- 主人公が家族の呼び声へ耐えられる根拠を、乗車直前の連絡と声が記憶を模倣しているという判断に置いた
- 車内の乗客が異なる年代に属する設定を、服装、乗車表、行方不明記録で段階的に説明した
- 怪異の規則を「返事または降車意思」「白い列への移動」「乗務員交代」の三段階に整理した
- 実在の会社、路線、事故を使用せず、運転操作や事故原因の詳細も避けた
- 残酷な身体描写ではなく、音、濡れた座席、車内表示、紙の記録、白い列で恐怖を構成した
- 「最後のエレベーター」の停止した閉鎖空間とは異なり、移動と周回を続ける車内を中心にした
- 「三回ノック」の過去への移動や記録改変とは異なり、異なる夜の運行が同時に重なる現象として差別化した
