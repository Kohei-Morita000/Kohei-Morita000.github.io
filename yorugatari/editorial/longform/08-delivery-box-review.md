## 維持した中核

### 起

- 離婚後、防犯設備を優先して七〇四号室へ入居する導入
- 新しい部屋番号プレートと寝室だけ異なる内装
- 午後十一時前後に届く未注文の受取通知
- 空の小型ボックスに残る消毒薬の匂いと、切り取られた住所・氏名

### 承

- 通知先が小型から中型へ変わり、繊維、食べ物、湿気、温度という生活痕跡が増える流れ
- 防犯映像が毎晩同じ時刻だけ乱れる事実
- 七〇四号室が旧管理人室と共用倉庫を改装した住戸であること
- 元管理人の息子・亮介が宅配ボックスを「小さい部屋」「奥の部屋」と呼んでいた記録
- 郵便物のコピーに、菜月が部屋を空けた時間が記録されている反転

### 転

- 管理会社が箱を止めた後、人が入れる大型ボックスから通知が来る展開
- 暗証番号入力後、内側から普通の咳払いが聞こえる場面
- 警察が毛布、水、食料、着替え、郵便物、現在と改装前の図面を発見
- 菜月の寝室が、旧図面で亮介の子ども部屋だった位置と重なる事実
- 最後の計測記録だけ帰室時刻が空欄である伏線

### 結

- 捜索後も亮介本人は見つからない
- 箱から回収した毛布と水が不足している
- 寝室の収納が旧子ども部屋側の壁へ動かされている
- 室内インターホンの付箋「今夜から、こっちにします」を最終文として維持

## 削減内容

- 四晩分の空箱確認を一つの経過へ統合
- 通知時刻、箱番号、温度測定の個別列挙を圧縮
- 管理台帳、旧図面、有馬の証言を一つの調査線へ整理
- 室内で起きる小さな変化と警察捜索の重複説明を削減
- 侵入経路や防犯設備の突破方法は引き続き具体化していない

## 伏線回収

| 伏線 | 回収 |
|---|---|
| 七〇四号室だけ新しいプレート | 旧管理人室を後から改装した住戸 |
| 寝室だけ異なる壁紙と床材 | 亮介の旧子ども部屋と同じ位置 |
| 小型から大型へ変わる箱 | 情報収集、滞在、生活拠点準備の段階 |
| 通知のたび部屋を空ける | 七〇四号室の無人時間の計測 |
| 一分だけ乱れる防犯映像 | 建物の古い不具合を利用した登録 |
| 最後の帰室時刻が空欄 | 亮介が箱から次の場所へ移った可能性 |
| 不足した毛布と水 | 生活用品が住戸側へ移された証拠 |
| 通知が止まる | 箱を使う必要がなくなったため |
