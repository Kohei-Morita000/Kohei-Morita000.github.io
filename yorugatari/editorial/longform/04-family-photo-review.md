## 圧縮方針

- 写真を年代順に確認する場面を統合
- 家系資料と火事の説明を、冬子が家族から除外された理由へ集中
- 顔認識と画像復元の検証回数を減らし、怪異の規則が一度で伝わる構成へ変更
- 家族の否定、撮影中の消失、名前を戻す場面を残し、終盤の緊張を維持

## 維持した中核

### 起

- 毎年の家族写真と一脚の空席
- 祖母の死で習慣が途切れる状況
- 椅子の裏の「冬子」
- 古い写真の少女と人数の不一致

### 承

- 空席が年ごとにカメラへ近づく
- 顔認識ソフトが冬子と空席を真奈として分類
- 空席を画像から消すと、叔父が写真と家族の記憶から消える
- 元画像を戻すと、叔父の存在と記憶も戻る
- 日記と新聞から、冬子が火事で祖母を救った後に家族の記録から消されたと判明

### 転

- 母と大伯父が冬子を否定し、母が前年の写真から消える
- 空席なしの撮影で親族が一人ずつ薄くなる
- 冬子がセルフタイマーの数字とともに前列へ近づく
- 真奈が椅子を戻し、家族全員に冬子の名前を口にさせる

### 結

- 冬子の記録を保存し、墓誌へ名前を加える相談を開始
- 祖母と冬子が同じ写真へ写る
- 撮影時になかった新しい空席を顔認識が真奈として100％分類
- 母が真奈を認識できなくなる
- 祖母の書き置き「次は、写真を見つけた人の番です」を最終文として維持

## 伏線回収

| 伏線 | 回収状況 |
|---|---|
| 祖母が空席の背を撫でる | 冬子の席を維持し、ほかの家族が消えるのを防ぐ行為として回収 |
| 椅子の裏の「冬子」 | 消された人物の名前と存在を戻す鍵として回収 |
| 写真人数が一人多い | 冬子が写真の家族数へ含まれていたことを示す |
| 母が冬子の好物だけ覚えている | 名前を否定しても感情に結びついた記憶が残ることを示す |
| 顔認識ソフトの真奈判定 | 最後の空席が真奈の席になる結末へ接続 |
| 空席を消すと叔父も消える | 空席がなければ別の家族を席へ取る規則として回収 |
| 祖母の注意書き | 写真、椅子、名前が記憶を維持する条件として回収 |
| カメラへ近づく空席 | 次に写真を見つける真奈へ役目が移ることを示す |
