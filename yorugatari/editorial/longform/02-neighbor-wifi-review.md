## 維持した中心要素

### 起

- 301号室と303号室の間に302号室がない違和感
- 相場より安い家賃と、すぐ転居できない現実的な事情
- 親切な管理人、塗り直された白壁、開けたまま使うよう勧められるクローゼット
- `302_たすけて`、黒い接続画面、三回のノック

### 承

- 午前3時02分だけ現れるSSIDと段階的な警告
- 用途不明の回路、壁際で強くなる電波、旧図面の302号室
- 管理人が生活時間を把握していた可能性
- 壁内から撮影された歴代住人と主人公の写真
- `302_いったね`から`302_みつかった`への変化

### 転

- 合鍵による玄関侵入とクローゼット裏の壁内通路
- 覗き穴、合鍵、生活記録、前居住者・高瀬真帆の端末
- 真帆が残した途切れた動画
- Wi-Fiの指示で非常ベルを作動させ、住人と警察を呼ぶ脱出行動

### 結

- 管理人の逮捕により人間の監視事件を一度解決
- ルーターと端末が無電源で、操作記録も五年前に止まっていた事実
- 真帆が302号室ではなく301号室の前居住者だった反転
- 警察署の証拠品から`301_たすけて`が現れる
- 未接続の端末への警告と、取調室の白壁からの三回ノック

## 圧縮内容

- 尾崎による翌朝の確認と生活時間の質問を一場面へ集約
- Wi-Fi警告の反復を、規則を示す三件へ限定
- 分電盤、電波強度、図面調査を短い証拠列へ整理
- 歴代住人の写真説明を撮影位置と代表例へ集約
- 壁内通路の物品列挙と警察捜査の説明を圧縮
- 刑事との会話を、無電源機器・真帆の居住歴・証拠品の再発へ集中
- 新居での再発は使用せず、証拠保全中の即時反転を維持

## 伏線回収

| 伏線 | 回収状況 |
|---|---|
| クローゼットを開けたまま使うよう勧める | 壁内扉へ管理人が接近しやすくする意図として回収 |
| 白く塗り直された壁 | 隠された壁内通路の入口として回収 |
| 用途不明の回路と強い電波 | 壁内の機器へ接続。ただし無電源という超常的反転を残す |
| 管理人が勤務時間を尋ねる | 侵入しやすい時間帯の確認だったと判明 |
| 補助錠 | 合鍵による侵入を一時的に止め、通路と警報を発見する時間を作る |
| 三回のノック | 壁内の警告から、警察署の白壁へ怪異が残る結末へ接続 |
| `302_みつかった` | 真帆の動画と現在の侵入危機を重ねる |
